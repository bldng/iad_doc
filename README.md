##IAD documentation skeleton builder

To run this, either clone this repository and run:

```bash
bash iad_doc.sh
```

or add it to your ~/.zshrc or ~/.bashrc:

```bash
chmod +x iad_doc.sh
cp -i iad_doc.sh ~/../../usr/local/bin
```

then add

```bash
alias iad_doc="iad_doc.sh"
```
